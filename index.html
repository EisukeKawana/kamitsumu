# 第3章　情報システムと社会

## 3.1 情報システムの定義
情報システムとは、データを収集・処理・保管・配布するための
相互に関連したコンポーネントのセットである。

## 3.2 主要なコンポーネント
- ハードウェア：物理的な装置
- ソフトウェア：プログラムと手順
- データ：事実と数値の集合
- ネットワーク：通信インフラ
- 人：最も重要な要素
